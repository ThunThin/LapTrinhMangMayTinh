Giới Thiệu Chung

Mình là Thiên, sinh viên năm cuối ngành Công nghệ Thông tin, chuyên ngành Cyber Security và Software Development.

Mình quan tâm đến cách xây dựng phần mềm an toàn, có cấu trúc rõ ràng và dễ mở rộng, thay vì chỉ tập trung vào việc "làm cho chạy được".

Trong quá trình học tập, mình nhận ra rằng viết được code chưa đủ — điều quan trọng hơn là hiểu vì sao hệ thống được thiết kế như vậy, cách các thành phần tương tác với nhau, và làm sao để phần mềm có thể duy trì, mở rộng và bảo mật trong môi trường thực tế.

Triết lý học tập

  • Learn by doing – Học thông qua thực hành và dự án thực tế, không chỉ đọc lý thuyết
  • Understand fundamentals – Nắm vững nền tảng (OOP, cấu trúc dữ liệu, networking, security concepts) trước khi tiếp cận framework
  • Write clean code – Code phải dễ đọc, dễ bảo trì và dễ mở rộng cho người khác

Định hướng nghề nghiệp

Mục tiêu của mình là trở thành một Software Engineer chuyên nghiệp, không chỉ biết viết code mà còn:

  • Hiểu kiến trúc hệ thống
  • Áp dụng best practices
  • Có tư duy giải quyết vấn đề rõ ràng và có hệ thống
  • Quan tâm đến yếu tố bảo mật ngay từ khâu thiết kế

Về blog này

Blog được tạo ra như một nhật ký học tập và chia sẻ kiến thức, nơi mình ghi lại:

  • Những gì đã học được
  • Những sai lầm từng gặp
  • Cách mình cải thiện tư duy lập trình qua từng giai đoạn

Mục tiêu của blog là giúp các bạn sinh viên tránh được những lỗi phổ biến, tiếp cận lập trình theo hướng bài bản, thực tế và hiệu quả hơn, từ đó rút ngắn con đường từ sinh viên đến developer thực thụ.

Hành Trình Từ Sinh Viên Đến Developer

// Từ sinh viên đến developer
class Developer {
    constructor(ten, chuyenNganh) {
        this.ten = ten;
        this.chuyenNganh = chuyenNganh;
    }
    
    hocTap() {
        return "Thực hành hàng ngày";
    }
    
    hieuBietNenTang() {
        return ["OOP", "Cấu trúc dữ liệu", "Networking", "Security"];
    }
    
    vietCodeSach() {
        return { deDoc: true, deBaoTri: true, deMoRong: true };
    }
    
    troThanhChuyenNghiep() {
        return { hieuKienTruc: true, apDungBestPractices: true, tuDuyGiaiQuyetVanDe: true };
    }
}
